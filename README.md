# Audio Analysis Agent 🎙️

An AI-powered audio processing agent that transcribes files, identifies speakers, analyzes sentiment, extracts topics, and features an interactive chatbot—all powered by the AssemblyAI API.

## ✨ Features

- **Asynchronous Transcription:** Fast processing for large audio files.
- **Speaker Diarization:** Accurately detects and labels different speakers.
- **Sentiment Analysis:** Tracks emotional shifts and tone per speaker turn.
- **Topic Detection:** Automatically categorizes themes with timestamp segmentations.
- **Interactive Chat (LEMUR):** Ask questions, summarize, or extract action items directly from the audio using LLM context.

## 🚀 Quick Start

### 1. Prerequisites
- Python 3.8 or higher
- An AssemblyAI API Key ([Get one here](https://assemblyai.com))
